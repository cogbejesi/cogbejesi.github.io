What's a Crabby Cabbi?

This team captured live streaming Capital Bike Share System Data and captured at 3 minute intervals of the data provided. This data was then stored on an relational database (SQL) on Amazon Web Services. We then were able to use this data to provide visual information information to our user. Additionally, we leveraged Leaflet to do a direct pull to render the current status of bikeshare locations.
